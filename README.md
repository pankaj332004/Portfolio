## ✨ Features

- **💎 Sleek Glassmorphism Design:** Modern design system built with custom CSS, vibrant gradient accents, and fine-tuned HSL tokens.
- **⚡ Lightning-Fast Performance:** Scaffolds on Vite + React 18, generating highly optimized production builds.
- **📱 Fully Responsive Layout:** Beautifully scaled across mobile, tablet, and desktop viewports.
- **🔄 Dynamic Category Filtering:** Real-time project filtering across **Full Stack**, **Frontend + AI**, and **Hackathon** classes.
- **📬 Functional Contact System:** Seamlessly integrates with Web3Forms to securely deliver message submissions straight to email inbox.
- **💫 Fluid Micro-interactions:** Smooth page transitions, parallax terminal motion, and on-scroll animation effects powered by Framer Motion.

## 🚀 Local Development Setup

Follow these simple commands to run the project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pankaj332004/PortFolio.git
   cd PortFolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root folder of the project:
   ```env
   # Get your free access key at https://web3forms.com
   VITE_WEB3FORMS_ACCESS_KEY="your_web3forms_access_key"
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser!

5. **Generate a production build:**
   ```bash
   npm run build
   ```
